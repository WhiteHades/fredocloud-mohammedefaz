# Keep FredoHub JavaScript-only across the monorepo and prioritise RBAC plus audit log

The assessment explicitly requires Next.js App Router with JavaScript and no TypeScript, while the backend stack is Express, Prisma, Socket.io, Cloudinary, and Railway. We will keep both `web` and `api` in JavaScript to follow the brief literally, reduce setup drag, and reserve complexity budget for the chosen advanced features: **Advanced RBAC** and **Audit log**, plus the full bonus set where external credentials allow live verification.
